# AI-Powered Data Analyst Assistant

---

## Project Overview
AI Data Analyst Chatbot is an AI-powered application that helps users analyze datasets through natural language conversations. Users can upload CSV or Excel files and interact with the chatbot to gain insights, identify trends, and receive recommendations without writing complex code or queries.

The chatbot leverages Large Language Models (LLMs) through the Groq API to provide data-driven analysis and support decision-making processes.

---

## Project Objectives

The objective of this project is to simplify data analysis by allowing users to interact with their datasets using natural language and receive meaningful insights automatically.

---

## Target Users
* Beginner Data Analysts
* Small Business Owners

---

## Technologies Used
* Python
* Streamlit
* Pandas
* LangChain
* Groq API
* Llama 4 Scout

---

## Benefits
* Simplifies data analysis for non-technical users.
* Reduces the time required for exploratory data analysis.
* Provides automated insights and recommendations.
* Supports data-driven decision-making.
* Enables interactive communication with datasets through natural language.
